# Claude Code Plugin Marketplace

This directory contains the marketplace configuration for distributing this skill as a Claude Code plugin.

## Installation

Users can install this skill directly in Claude Code using:

```
/plugin marketplace add BurnWang/apple-podcast-downloader
/plugin install podcast-downloader
```

## Marketplace Configuration

The `marketplace.json` file defines:
- Plugin metadata (name, description, version)
- Author information
- Plugin source location
- Categories and tags for discovery
